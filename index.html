Cloud Computing

Rizka Nayara | 211011401190 | 04TPLP010

Cloud Computing adalah teknologi yang memungkinkan pengguna untuk mengakses server, penyimpanan, dan aplikasi melalui internet tanpa perlu memiliki perangkat fisik sendiri.

Pertemuan 01

𝜗ৎ Instalasi Sistem Operasi Debian 𝜗ৎ

Pada pertemuan ini dilakukan instalasi sistem operasi Debian sebagai dasar dalam membangun server. Debian dipilih karena stabil dan banyak digunakan dalam lingkungan cloud computing.

  • Menyiapkan file ISO Debian.
  • Membuat Virtual Machine (RAM 2GB, Storage 20GB).
  • Booting dan memulai instalasi sistem.
  • Konfigurasi hostname dan domain server.
  • Membuat user root dan user biasa.
  • Instalasi paket sistem dasar.
  • Menyelesaikan instalasi hingga berhasil login.
Pertemuan 02

𝜗ৎ Konfigurasi IP Address 𝜗ৎ

Konfigurasi IP Address dilakukan untuk memberikan alamat tetap (static IP) pada server agar mudah diakses dalam jaringan.

  • Cek interface dengan ip a
  • Edit file /etc/network/interfaces
  • Masukkan IP Address, Netmask, Gateway
  • Restart networking
  • Uji koneksi dengan ping
  • Pastikan server terhubung jaringan
Pertemuan 03

𝜗ৎ Instalasi Web Server Apache 𝜗ৎ

Pada tahap ini dilakukan instalasi Apache sebagai web server untuk menjalankan website pada sistem Debian.

  • Install Apache: apt install apache2
  • Cek status: systemctl status apache2
  • Akses melalui browser menggunakan IP server
  • Edit file di /var/www/html
  • Deploy file HTML
  • Uji akses website
