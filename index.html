LAPORAN PRAKTIKUM

CLOUD COMPUTING

────────

Cloud computing atau komputasi awan adalah teknologi yang memungkinkan pengguna untuk mengakses, menyimpan, dan memproses data serta aplikasi melalui internet tanpa perlu memiliki infrastruktur fisik yang kompleks.

Pertemuan 01

Instalasi Sistem Operasi Debian

Persiapan Lingkungan

  • Penyediaan File ISO Debian Stable.
  • Setup Virtual Machine (RAM 2GB, Storage 20GB).

Eksekusi Instalasi

  • Konfigurasi Hostname dan Domain.
  • Manajemen User (Root & Standard User).
  • Finishing instalasi sistem dasar.
Pertemuan 02

Konfigurasi IP Address

Network Analysis

  • Cek interface aktif dengan ip address show.

Konfigurasi Statis

  • Edit /etc/network/interfaces.
  • Input Address, Netmask, dan Gateway secara manual.
  • Restart servis networking untuk aktivasi.
Pertemuan 03

Web Server Debian

Implementasi Apache

  • Instalasi via apt install apache2.
  • Verifikasi status layanan Apache aktif.

Deployment Konten

  • Upload file HTML ke /var/www/html/.
  • Uji akses via HTTP melalui IP Address Server.
